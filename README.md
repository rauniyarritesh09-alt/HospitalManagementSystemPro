# 🏥 Hospital Management System

A Java Swing Desktop Application for managing Hospital Operations.

---

## 📌 Features

- Login Authentication
- Dashboard
- Patient Management
- Doctor Management
- Appointment Management
- Billing Management
- Search Records
- View Records
- PDF Export
- Reports
- SQLite Database
- Professional GUI
- Icons Support

---

## 🛠 Technologies Used

- Java
- Java Swing
- SQLite
- JDBC
- Maven
- IntelliJ IDEA
- iText PDF
- LGoodDatePicker
- Git & GitHub
